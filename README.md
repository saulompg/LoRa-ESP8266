# LoRa-ESP8266

> Comunicação entre dispositivos ESP8266 através da tecnologia de radiofrequência LoRa RA-02 SX1278. 

## CONEXÃO LoRa RA-02 - ESP8266
- NSS   -  GPIO15  :  D8
- MOSI  -  GPIO13  :  D7
- MISO  -  GPIO12  :  D6
- SCK   -  GPIO14  :  D5
- RST   -  GPIO2   :  D4
- DIO0  -  GPIO5   :  D1
